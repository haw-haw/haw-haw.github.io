背景

MySQL server

  3. / 区一般都不大，而且不推荐放数据
  4. 所以但凡正常 MySQL server，都存在 Data 目录放真正数据分区的需求
  5. 如果刚开始时没有设置好 Data 目录在数据分区，那么后来都有迁移 Data 目录到数据分区的需求
